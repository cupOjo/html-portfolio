General Information

  • Founded: 2016 by Dominic Williams.
  • Headquarters: Zurich, Switzerland.
  • Purpose: To create a decentralized computing platform for running secure software without relying on centralized services.
  • Flagship Product: The Internet Computer, a blockchain-based platform designed to extend the public internet so it can host backend software.
